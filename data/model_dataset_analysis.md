# Model and Dataset Analysis

## DeepSeek-R1-Distill-Qwen-1.5B Model

The DeepSeek-R1-Distill-Qwen-1.5B model is a distilled version of the larger DeepSeek-R1 model, specifically designed for efficient reasoning tasks. Key characteristics include:

- **Architecture**: Based on Qwen2.5 architecture
- **Size**: 1.5 billion parameters (1.788 params according to HuggingFace)
- **Training Approach**: Distilled from the larger DeepSeek-R1 model using samples generated by it
- **Tensor Type**: BF16
- **Performance**: Despite its compact size, it shows strong reasoning capabilities:
  - AIME 2024: 28.9% pass@1, 52.7% cons@64
  - MATH-500: 83.9% pass@1
  - GPQA Diamond: 33.8% pass@1
  - LiveCodeBench: 16.9% pass@1
  - CodeForces rating: 954

The model was trained via large-scale reinforcement learning (RL) techniques, which allowed it to develop strong reasoning capabilities. The distillation process transferred reasoning patterns from the larger model to this more compact version, making it suitable for resource-constrained environments while maintaining good performance.

## Reference Model: roberta-large-self-disclosure-sentence-classification

This is the model described in the paper "Reducing Privacy Risks in Online Self-Disclosures with Language Models" that we're using as a reference. Key characteristics include:

- **Base Architecture**: Fine-tuned from FacebookAI/roberta-large
- **Task**: Binary sentence-level classification to determine if a sentence contains self-disclosure (1) or not (0)
- **Performance**: 88.6% accuracy on the classification task
- **Usage**: Designed specifically for privacy risk detection in user-generated content
- **License**: Creative Commons Attribution-NonCommercial

The model is used to classify whether a given sentence contains disclosure or not, serving as the detection component in the two-part system (detection and abstraction) described in the paper.

## Dataset: reddit-self-disclosure

The dataset used for training and evaluating the reference model has the following characteristics:

- **Format**: conll IOB2 format (Inside-Outside-Beginning tagging scheme)
- **Annotation**: Each instance in batches 1-8 is annotated by one annotator, while instances in batches 9 and 10 are annotated by two annotators followed by adjudication
- **Content**: Contains Reddit posts with annotated self-disclosure spans
- **Size**: Between 1M-10M samples according to HuggingFace
- **License**: cc-by-nc-sa-4.0
- **Usage Restrictions**: Limited to research purposes only, redistribution prohibited without authors' permission, compliance with Reddit's policy mandatory

The dataset is designed to identify spans of text that contain self-disclosures, which are then categorized according to the taxonomy of 19 self-disclosure categories (13 demographic attributes and 6 personal experiences) mentioned in the paper.

## Comparative Analysis for Adaptation

To adapt the DeepSeek-R1-Distill-Qwen-1.5B model for privacy risk detection, we need to consider:

1. **Architecture Differences**: 
   - DeepSeek-R1-Distill-Qwen-1.5B is a generative model based on the Qwen architecture
   - The reference model is a classification model based on RoBERTa

2. **Task Adaptation**:
   - We need to adapt the generative capabilities of DeepSeek-R1-Distill-Qwen-1.5B for the classification task
   - Alternatively, we could frame the task as a generative one where the model identifies and explains privacy risks

3. **Training Approach**:
   - Fine-tuning on the reddit-self-disclosure dataset
   - Few-shot learning using examples from the dataset
   - Potential distillation from the reference model

4. **Evaluation Strategy**:
   - Direct comparison with the reference model on the same test set
   - Measuring accuracy, precision, recall, and F1 score for the classification task
   - Evaluating the quality of explanations if using a generative approach

The compact size of DeepSeek-R1-Distill-Qwen-1.5B (1.5B parameters) makes it suitable for deployment in resource-constrained environments, which could be advantageous for real-time privacy risk detection in user-generated content.
